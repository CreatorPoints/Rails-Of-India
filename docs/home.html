Overview

Getting Started

Rails of India is a 1:1 scale train simulation that focuses on accurate routes, signaling, and station detail. These docs are structured to help you move from download to contribution without guesswork.

What is the Core?

The Core is the base game package. It ships with a starter route, five stations, one EMU, and baseline vegetation. Although it looks small, it is data-heavy because it relies on GIS sources like NASA Earthdata, OpenStreetMap, and ISRO Bhuvan. The Core is updated for stability and compatibility.

What are Patches?

Patches are update packages that extend the Core. Three types exist: Minor Patch (.roimip), Major Patch (.roimap), and Grand Patch (.roigp). Use them to stay current without redownloading the full base build.

What are the system requirements?

Final requirements are still being refined. For now, assume hardware above the Current Specs listed on the download page. Optimization is a priority even though the game runs on Unreal Engine 5.

What engine does the game run on?

The project runs on Unreal Engine 5.7.4 to target realism while keeping performance front and center.

Formats

Patch Formats

Patches are versioned packages. Use the correct type for your update path.

Type Extension Use Case
Minor Patch .roimip Incremental updates applied in order.
Major Patch .roimap Bundled patch range, replaces multiple minors.
Grand Patch .roigp Milestone release with broader changes.
Info: If you need older versions of patches, head to our Archive (If the earlier versions were ok to keep)

FAQ

Where do I find the latest patch?

Check the download page for the newest patch list. The table is weekly updated.

Do I need every patch?

Yes (or your wish), but make sure you follow the rules listed in Download page, or else it will break the game.

Will this run on low-end hardware?

Absolutely, as it is developed in hardware older than 50% of the people in world, but still you need good hardware for good experience.
